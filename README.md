# Delivery Delay Dashboard - Excel Project

## Project Objective
To analyze delivery performance and track on-time and delayed orders using Excel.

## Tools Used
- Microsoft Excel

## KPIs
- Total Orders
- On-Time Deliveries
- Delayed Deliveries
- On-Time Delivery Percentage

## Dashboard Features
- Pivot Tables
- Charts
- Slicers for date and region analysis
  
## Configuration Details
- Data cleaned using Excel formulas
- Pivot Tables used for KPI calculation
- Charts created for visual analysis
- Slicers added for dynamic filtering
- Dashboard designed in a single worksheet

## Conclusion
This dashboard helps monitor delivery efficiency and identify delay issues for improvement.
